# Swarm Engineering Contract

This repository is part of the Swarm autonomous AI system.

All coding agents must follow the rules below.

## Protected System Areas

The following directories are READ ONLY:

```
core/
contracts/
tests/
```

Do not modify these paths.

## Allowed Edit Zones

```
agents/
pipelines/
scripts/
```

## Coding Rules

1. Never rewrite large files.
2. Only implement minimal changes.
3. Preserve architecture.
4. Always add tests for new features.
5. Prefer patch-based changes.

## Workflow

1. Propose a plan.
2. Generate a patch.
3. Run tests.
4. Submit change.

Stability is the highest priority.
